    Lesson 5: <Record>


    The <Record> verb records the callerâ€™s voice and returns the URL of a file containing the recording. Take it for a spin!

    Attribute Name Allowed Values Default Values
    action relative or absolute URL current document URL
    method GET, POST POST
    timeout positive integer 5
    finishOnKey any digit, #, * 1234567890*#
    maxLength integer greater than 1 3600 (1 hour)
    transcribe true, false false
    transcribeCallback relative or absolute URL none
    playBeep true, false true

    action:

    The action attribute is a URL that will handle the results of the <Record> verb. This URL can be hit with either a GET or a POST request. <Record> will give this URL another URL that contains the audio file in the RecordingUrl parameter. TwiML will then be fetched from this address and executed. Note: In general, any TwiML after a <Record> verb is unreachable. The only exception to this is if your recording is silent, the action URL will not be fetched and execution will continue to the next verb.


    method:

    The method attribute controls whether Twilio sends a GET or a POST to the action URL.


    timeout:

    The <Record> verb will exit after this many seconds have elapsed.

    finishOnKey:

    The <Record> verb will exit after this key has been pushed.

    numDigits:

    The <Record> verb will exit after this many digits have been entered.

    maxLength:

    The <Record> verb will exit after this many seconds have elapsed.

    transcribe:

    If this attribute is true, Twilio will attempt to convert your recording to text. Note that this feature costs extra.

    transcribeCallback:

    The transcribeCallback attribute is used in conjunction with the transcribe attribute. When Twilio has finished transcribing your recording, it will make a POST request to this URL along with some parameters, the most interesting of which is TranscriptionText, which contains the actual transcription.

    playBeep:

    If this attribute is set to true, a beep will play to indicate the start of recording.
