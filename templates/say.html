    Lesson 2: <Say>


    The <Say> verb converts text into speech that will be played in your call. Try it out!

    Attribute Name Allowed Values Default Values
    voice man, woman man
    language en, en-gb, es, fr, de en
    loop integer >=0 1

    Voice:

    The voice attribute controls whether the text is read with a male voice or a female voice.


    Language:

    The language attribute controls what languageâ€™s accent and pronunciations will be used to read the text.


    Loop:

    The loop attribute controls how many times the text is repeated. A value of 0 will cause the text to loop until the call is hung up.
