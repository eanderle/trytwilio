    Lesson 3: <Play>


    The <Play> verb plays the file at the given url. Many different audio formats are supported--give it a shot!

    Attribute Name Allowed Values Default Values
    loop integer >= 0 1

    Loop:

    The “loop” attribute controls how many times the audio is played. A value of 0 will cause the audio to loop until the call is hung up.
